# Skill: standup

Use this to generate a team status summary.

1. Read `data.json` for each agent in `team/`.
2. Count tasks by status across all agents:
   - `todo`: waiting for PM to delegate
   - `scoped`: ready for agents to pick up
   - `in_progress`: actively being worked
   - `review`: awaiting review
3. Call out any blocked tasks or overdue items.
4. List the top 1–2 priorities for today.
5. Keep the summary under 20 lines.
